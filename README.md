# Area and perimeter calculator app built in Rust

This simple CLI app calculates the area and/or the perimeter of the selected shape given the required parameters.

## Supported shapes:

- Circle
- Ellipse
- Parallelogram
- Rectangle
- Regular polygon
- Sector
- Square
- Trapezoid
- Triangle

## Development:

```bash
cargo run
```

### Build:

```bash
cargo build --release
```
